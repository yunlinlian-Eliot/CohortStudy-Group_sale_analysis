# Data Dictionary

This directory contains the anonymized raw dataset used for the Group Sales Analysis.

### File: `Raw_data_Anony.csv`

| Column Name | Description |
| :--- | :--- |
| **Customer_ID** | Unique identifier for each customer (Masked). |
| **Purchase_Date** | The date of the transaction (YYYY-MM-DD). |
| **Amount_Ex_GST** | Transaction value excluding GST (Standardized to 1000/-1000/0). |
| **Treatment_Name** | Type of aesthetic service provided (Anonymized). |

> **Note:** All PII (Personally Identifiable Information) has been removed to comply with PDPA standards.
